# Avnish Sinha's Portfolio Website

Welcome to my personal portfolio website, hosted on GitHub Pages. You can view it live at [avnishsinha.github.io](https://avnishsinha.github.io/).

## 📂 Project Structure

The repository consists of the following directories and files:

```
📦 avnishsinha.github.io
 ┣ 📂 assets              # Various assets used in the project
 ┣ 📂 css                 # Stylesheets for the website
 ┣ 📂 images              # Image assets
 ┣ 📂 js                  # JavaScript files for interactivity
 ┣ 📂 typewriting-master  # Typewriting effect implementation
 ┣ 📜 LICENSE.txt         # License file
 ┣ 📜 README.md           # This README file
 ┣ 📜 README.txt          # Another README format
 ┣ 📜 index.html          # Main HTML file
```

## 🚀 Features

- **Modern & Responsive Design** – The website is optimized for different screen sizes.
- **Typewriting Effect** – Adds an engaging dynamic text animation.
- **Custom Styling** – Uses CSS and SCSS for enhanced styles.
- **JavaScript Enhancements** – Interactive elements powered by JavaScript.
- **GitHub Pages Deployment** – Automatically deployed using GitHub Pages.

## 🛠️ Technologies Used

- **HTML** – Structuring the content
- **CSS / SCSS** – Styling the website
- **JavaScript** – Adding interactivity
- **GitHub Pages** – Hosting the website

## 📦 Deployment

The website is continuously deployed using GitHub Pages. Any changes pushed to the `main` branch are automatically published.
